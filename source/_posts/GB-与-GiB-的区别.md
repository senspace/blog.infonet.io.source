# 计算机、单位、二进制

- **KB** 是 kilobyte 的缩写，指的是千字节。

| | |
|:--:|:---------:|
| 1KB | 1000Byte |
| 1MB | 1000KB   |
| 1GB | 1000MB   |


- **KiB** 是 kilo binary byte 的缩写，指的是一千个二进制字节。

| | |
|:----:|:--------:|
| 1KiB | 1024Byte |
| 1MiB | 1024KiB  |
| 1GiB | 1024MiB  |
