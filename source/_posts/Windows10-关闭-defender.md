# 1. 步骤
1. 快捷键 **WIN+R** 打开运行工具
2. 输入 **gpedit.msc** 回车
3. 计算机配置 -> 管理模板 -> Windows 组件 -> Windows Defender 防病毒程序 -> 关闭 Windows Defender 防病毒程序
4. 双击进入选择已启用确定即可。
